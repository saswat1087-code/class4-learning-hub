## 🌟 Overview

**Class 4 Learning Hub** is an engaging, gamified educational platform designed specifically for 9-10 year old students. Powered by Google's Gemini AI, it provides personalized tutoring, interactive quizzes, progress tracking, and a friendly AI assistant - all wrapped in a colorful, kid-friendly interface.

### ✨ Key Features

| Feature | Description |
|---------|-------------|
| 🤖 **AI Exam Buddy** | 24/7 AI tutor that answers questions, explains concepts, and helps with homework |
| 📚 **Interactive Syllabus** | Browse chapters across 4 subjects with AI-powered summaries |
| 🏆 **Practice & Quizzes** | Generate custom quizzes, track scores, and earn points |
| 📊 **Progress Tracking** | Visual analytics, badges, streaks, and achievements |
| 👨‍👩‍👧 **Parent Dashboard** | Monitor progress, view reports, and track learning patterns |
| 🎮 **Gamification** | Points system, levels, badges, and daily challenges |
