# Desktop Management

## Content
The Desktop is the main screen of your computer. Proper desktop management helps you work efficiently.

**Desktop Components:**

- **Wallpaper/Background**: The picture behind your icons
- **Icons**: Small pictures representing programs, files, or folders
- **Taskbar**: The bar at the bottom showing open programs
- **Start Button**: Opens the Start Menu
- **System Tray**: Shows time, volume, and running background programs

**Organizing Your Desktop:**

- Create folders to group similar files
- Delete unnecessary shortcuts
- Arrange icons by name, size, or date
- Use Recycle Bin to delete files permanently

**File Management:**

- **Create Folder**: Right-click → New → Folder
- **Rename**: Right-click → Rename
- **Move**: Drag and drop to new location
- **Delete**: Drag to Recycle Bin or press Delete key
- **Restore**: Open Recycle Bin and click Restore

**Taskbar Settings:**

- Pin frequently used programs to taskbar
- View open programs as buttons
- Show desktop button (far right corner)

## Key Points
- Desktop is the main screen with icons and taskbar
- Use folders to organize files on desktop
- Deleted files go to Recycle Bin first
- Right-click gives you options menu
- Taskbar shows running programs

## Vocabulary
- **Desktop**: Main computer screen
- **Icon**: Small picture representing a program or file
- **Taskbar**: Bar showing open programs
- **Recycle Bin**: Temporary storage for deleted files
- **Shortcut**: Quick link to a program or file
- **Folder**: Container for organizing files

## Fun Fact
The computer desktop metaphor (icons, folders, trash can) was invented at Xerox PARC in the 1970s. Apple and Microsoft later made it popular! 🖥️
