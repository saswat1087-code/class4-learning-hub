# Editing in MS Word 2013

## Content
Editing means making changes to a document. MS Word 2013 provides many tools to edit text easily.

**Basic Editing Operations:**

- **Typing**: Enter text using the keyboard
- **Deleting**: Use Backspace (delete left) or Delete (delete right)
- **Selecting Text**: Click and drag to highlight text
- **Cut**: Remove text and save to clipboard (Ctrl + X)
- **Copy**: Duplicate text to clipboard (Ctrl + C)
- **Paste**: Insert text from clipboard (Ctrl + V)
- **Undo**: Reverse last action (Ctrl + Z)
- **Redo**: Repeat last undone action (Ctrl + Y)

**Finding and Replacing Text:**

1. Press Ctrl + F to open Find dialog box
2. Type the word you want to search
3. Press Ctrl + H to open Replace dialog box
4. Type the new word and click Replace All

**Working with Multiple Documents:**

- Switch between open documents using taskbar
- Copy content from one document to another

## Key Points
- Cut, Copy, and Paste are essential editing commands
- Ctrl + Z undoes your last action
- Ctrl + Y redoes an undone action
- Use Find (Ctrl+F) to search for words
- Use Replace (Ctrl+H) to change words

## Vocabulary
- **Clipboard**: Temporary storage area for cut/copied items
- **Cut**: Remove text and save to clipboard
- **Copy**: Duplicate text to clipboard
- **Paste**: Insert clipboard content
- **Undo**: Reverse last action
- **Find**: Search for specific text

## Fun Fact
The Cut, Copy, and Paste commands were invented by Larry Tesler at Xerox PARC in 1974. Apple later made them popular on the Macintosh! ✂️📋
