# Large Numbers (Up to 10,00,000)

## Content
In Class 4, we learn about numbers up to 10,00,000 (Ten Lakhs). Let's understand place value and how to read large numbers.

**Place Value Chart (Indian System):**

| Crores | Lakhs | Thousands | Hundreds | Tens | Ones |
|--------|-------|-----------|----------|------|------|
|   1    |   0   |    0      |    0     |  0   |  0   |

**Reading Large Numbers:**
- 1,00,000 = One Lakh
- 5,00,000 = Five Lakhs
- 10,00,000 = Ten Lakhs (One Million)
- 45,67,890 = Forty-five Lakhs Sixty-seven Thousand Eight Hundred Ninety

**Comparing Numbers:**
- Greater Than (>) : 5,67,890 > 4,56,789
- Less Than (<) : 4,56,789 < 5,67,890
- Equal To (=) : 5,67,890 = 5,67,890

**Ordering Numbers:**
- Ascending Order (Smallest to Largest): 12,345 → 23,456 → 34,567
- Descending Order (Largest to Smallest): 34,567 → 23,456 → 12,345

**Expanded Form:**
4,56,789 = 4,00,000 + 50,000 + 6,000 + 700 + 80 + 9

## Key Points
- Numbers can go up to 10,00,000 (Ten Lakhs)
- Place values: Ones, Tens, Hundreds, Thousands, Lakhs
- Use commas after every two digits: 4,56,789
- Use > for greater than, < for less than, = for equal to
- Ascending = smallest to largest, Descending = largest to smallest

## Vocabulary
- **Place Value**: The value of a digit based on its position
- **Ascending**: Arranging from smallest to largest
- **Descending**: Arranging from largest to smallest
- **Greater Than**: When one number is larger than another (>)
- **Less Than**: When one number is smaller than another (<)
- **Expanded Form**: Writing a number as a sum of its place values
- **Lakh**: Indian term for 1,00,000

## Fun Fact
The largest number name is "centillion" which has 303 zeros! That's more than all the grains of sand on Earth! 🔢
