# Storage and Memory Devices

## Content
Data is raw facts and figures. Information is processed data that has meaning. RAM (Random Access Memory) is volatile memory that temporarily stores data while the computer is on. ROM (Read Only Memory) is non-volatile memory that stores permanent instructions. Secondary storage devices like hard disks, pen drives, and memory cards store data even when the computer is off.

Computers need different types of memory to work properly:
- **Primary Memory**: RAM and ROM (inside the computer)
- **Secondary Storage**: Hard drives, SSDs, pen drives (can be removed)

## Key Points
- Data = Raw facts, Information = Processed data that makes sense
- RAM is temporary memory (volatile) - forgets when power is off
- ROM is permanent memory (non-volatile) - remembers everything
- Hard disks and pen drives are secondary storage devices
- 1 KB = 1024 Bytes, 1 MB = 1024 KB, 1 GB = 1024 MB

## Vocabulary
- **Data**: Raw facts and figures without meaning
- **Information**: Processed data that has meaning
- **RAM**: Random Access Memory - temporary storage
- **ROM**: Read Only Memory - permanent storage
- **Storage**: Where data is kept for later use
- **Memory**: Computer's temporary workspace
- **Volatile**: Data that disappears when power is off
- **Non-volatile**: Data that stays even when power is off

## Fun Fact
The first hard disk drive, made in 1956, weighed over 1 ton and could only store 5MB of data! Today's tiny microSD cards can store 1,000,000 MB (1TB) - that's 200,000 times more storage! 💾
